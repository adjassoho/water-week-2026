# Deploying Water Week 2026 to Vercel

This project is built with Next.js and is optimized for deployment on Vercel.

## Prerequisite

1.  Create an account on [Vercel](https://vercel.com).
2.  Install the [Vercel CLI](https://vercel.com/docs/cli) (optional, for command line deployment).

## Option A: Deploy via GitHub (Recommended)

1.  Push your code to GitHub (already done).
2.  Go to your Vercel Dashboard.
3.  Click **"Add New..."** -> **"Project"**.
4.  Import the `water-week-2026` repository.
5.  Vercel will automatically detect the Next.js framework.
6.  Click **"Deploy"**.

## Option B: Deploy via Vercel CLI

1.  Run the following command in your terminal:
    ```bash
    npx vercel
    ```
2.  Follow the prompts to link the project to your Vercel account.
3.  Use default settings (just press Enter).
